# CAD Client Experience and Accessibility Model

> **Status:** Compatibility index; not a separate normative model.

This filename is retained to preserve existing repository links created before client experience and accessibility were separated into distinct responsibilities.

The authoritative documents are:

- [Client Experience Model](client-experience-model.md)
- [Accessibility and Inclusive Interaction Model](accessibility-and-inclusive-interaction-model.md)

Client experience and accessibility are related, but they are not interchangeable.

The Client Experience Model defines how an interface complements the user's role, preserves attention and context, communicates state and outcomes, supports recovery, and avoids becoming an obstacle to the work.

The Accessibility and Inclusive Interaction Model defines how supported people can perceive, understand, navigate, operate, and complete that work with appropriate independence, safety, privacy, and assurance.

New architecture references should link directly to the applicable authoritative document rather than this compatibility index.
