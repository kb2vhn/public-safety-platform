# CAD Operational Workstation Examples

> **Status:** Non-production examples and validation fixtures only

## Purpose

Reserve a governed location for machine-readable workstation profiles, systemd
service and socket examples, local IPC contracts, package manifests, network
profiles, and recovery fixtures.

An example in this directory must:

- Identify all placeholders.
- Avoid real credentials, addresses, protected information, and production
  identifiers.
- Use **workstation component** for a locally supervised CAD process or surface.
- Reserve **module** for a top-level Iron Signal Platform module such as CAD.
- State which architecture document it illustrates.
- Remain visibly non-production until a separate implementation and acceptance
  process promotes an artifact into an authoritative implementation tree.

The absence of a production-ready example does not weaken the governing
architecture.
