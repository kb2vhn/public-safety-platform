# DNP Branding Assets

This directory contains current DNP project-identity assets owned and maintained
by Iron Signal Systems.

## Primary Badge

```text
dnp-platform-badge.png
```

Recommended alt text:

```text
DNP — Domain-Neutral Platform Foundation badge
```

Do not use the image as the only source of the project name. Textual interfaces,
documentation, release metadata, and accessibility surfaces must also identify
the project as **DNP — Domain-Neutral Platform**.

Historical branding should remain with historical evidence when changing it
would misrepresent the accepted artifact or the identity under which it was
produced.
